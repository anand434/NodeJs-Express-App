# API App
Simple Node-Express-Mongog app to create apis. 

## Description
This is a simple rest api app build using Node and Express.
Implemenatioa of CRUD operations using GET, PUT, POST, PATCH, DEL.

### Executing program
* Clone/Download Repo.
* In command line type to run the app.
```sh
npm start
```
